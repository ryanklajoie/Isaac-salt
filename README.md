# Isaac-salt
Give weight of meat and get salt percentage back in grams.
Enter the weight of meat in either pounds or kilograms. Pick a percentage of the weight for the salt e.g. 1.1 or 1.3%. Get the amount of salt needed to season the meat back in grams. There is a button for percentages that are used very common like the 1.2% salinity that is standard in the kitchen. Users can add different percentages if they have different needs e.g. bacon needs a much higher percentage of salt by weigh so the user could add "bacon" as a preset and use it for each batch of pork belly.
